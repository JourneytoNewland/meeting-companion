> 一个具有"灵魂"的虚拟会议引导数字人，支持用户自定义原则、流程和人格特征。

## 功能特性

### 核心功能

- **🧠 数字人灵魂配置** - 自定义名称、头像、声音风格、价值观
- **⚖️ 会议原则管理** - 6种可配置原则
- **🗺️ 会议流程模板** - 5种预设模板
- **🎙️ 会议录音** - 实时录音
- **🔊 语音播报** - TTS中文语音
- **📝 会议纪要** - 实时记录导出
